FlexBox Tutorials

What is FlexBox?

Idea behind flexbox is to give the container an ability to alter the items present inside it, in order to fill the available space in best way. Flexbox is a whole module and not a single property, it involves a lot of things including its whole set of properties. Some of them are meant to be set on the container (parent element, known as “flex container”) whereas the others are meant to be set on the children (said “flex items”).

A flex layout looks like this one:
( Remember: Items can be laid out either along the main axis or along cross axis )

Why FlexBox?

It is very helpful and dev favorite and specially used in designing a website for small screened devices.

flex-direction

As the name itself says, it is related to flex direction. So if you recall, about our discussion on flex layout( see it here ), we said there are two axis, one is main axis (row) and other one is cross axis (column). We can arrange the children(flex-items) along these axis in any form i.e. along row or along column or in reverse order in a row as well as in column.
So lets see flex-direction in action:

flex-direction:'row |column | row-reverse | column-reverse'

.flex-container-1{
display:flex;
flex-direction : row;
border:2px dashed blue;
padding:20px;
